# Codeleap Posts

<br />

> The Codeleap project consists of a complete post CRUD with navigation routes, animations, responsive design, and pagination.

<br />

## Technologies/Libraries

- React (vite)
- Typescript
- Styled Components
- Redux Toolkit
- Redux Thunk
- Axios
- Moment
- React Router Dom
- React Toastify
- Svg Loaders

<br />

## Features

- Login/Logout
- Routes
- List Posts
- Create new Posts
- Update Posts
- Delete Posts
- Pagination
- Responsive Design

<br />

## How to Start the Project:

<br />

Clone the repository:

```sh
git clone https://github.com/matheuspedrosoo/codeleap.git
```

Enter the downloaded directory:

```sh
cd codeleap
```

Install the dependencies:

```sh
yarn ou npm install
```

Run the project:

```sh
yarn dev
```

or

```sh
 npm run dev
```

Access the application:

```sh
 http://localhost:5173/
```
